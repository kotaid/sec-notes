# sec-notes

I'm documenting my cybersecurity journey publicly — and this is post #1.

I come from a full stack web development background. This year I made the decision to transition into cybersecurity, specifically as a SOC analyst.

Since then I've been studying hard — A+, Network+, Security+, TCM SOC 101, PSAA, ISC2 CC, SC-200, and Splunk SPLK-5001. Along the way I've been keeping detailed notes on everything I've learned.

Instead of keeping those notes to myself, I'm going to start sharing them publicly — formatted as proper writeups on GitHub.

My goal is simple: document the journey, give back to others who are on the same path, and show my work to the world.

First writeup is live: The OSI Model from a SOC analyst's perspective — not just the theory, but why it actually matters in security.

If you're also breaking into cybersecurity, or just getting started — follow along. This is for you.
